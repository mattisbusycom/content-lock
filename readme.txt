Show or hide some or all of your content by wrapping it in convenient shortcodes!

== Description ==

Show or hide content with a simple to use shortcode system.  You can access or restrict content for:

* Logged in/out users
* different user types, (author, administrator, etc.)
* different user roles and capabilities
* URL Keys
* password entry (good to show teasers, etc)
* show/hide clickable links

== Installation ==

1. Installation is ready upon activation of plugin.  Simply use the shortcodes inside your pages and posts.
